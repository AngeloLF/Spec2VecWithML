# Spec2VecWithML
Machine Learning Project, mixing 4 old repositories : 
  * `SpecSimulator` : capable to create simulate pairs of Spectrum / Spectrogram for multiple observatory
  * `Spec2vecModels` : containt some ML models (CNNs, ViTs, ...) to predict a spectrum from a spectrogram
  * `Spec2vecAnalyse` : for apply and analyse ML models
  * `ExtractAtmo` : for extract atmosphere parameters from predicted spectrum
