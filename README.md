# Spec2VecWithML
Machine Learning Project, mixing 4 old repositories : 
  * `SpecSimulator` : capable to create simulate pairs of Spectrum / Spectrogram for multiple observatory -> **specSimulator**
  * `Spec2vecModels` : containt some ML models (CNNs, ViTs, ...) to predict a spectrum from a spectrogram -> **models**
  * `Spec2vecAnalyse` : for apply and analyse ML models -> **applies** & **analyses**
  * `ExtractAtmo` : for extract atmosphere parameters from predicted spectrum -> **extractAtmos**
  * `TelebotPerso` : for create batch with ccin2p3 and create bot (Telegram and Discord) -> **jobAndBots**

This repo containt also a folder **Spectractor**. It's a modified version of original algo here :
 * https://github.com/LSSTDESC/Spectractor
